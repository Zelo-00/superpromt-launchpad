# SuperPromt — Дизайн-система «Молтен-Слэб» (Molten Slab)

> Единый источник правды по визуалу веб-приложения. Реализуется в `web/frontend/index.html`.
> Принцип: НЕ шаблонный дашборд, а **точный измерительный инструмент, выкованный из металла**.
> Сырой промт под жаром доводится до точной спецификации (наследие «лавы» бренда) + эстетика
> машинного калибра (мера, точность). **Строго без эмоджи** — только кастомные inline-SVG.

## 1. Концепция и настроение
- Обсидиан-тёмный фон, молтен-акцент (расплавленный металл), холодная сталь для данных/цифр.
- Ощущение: высокоточный прибор (штангенциркуль/манометр), а не «карточки с эмоджи».
- Ключевой мотив-подпись: **шкала делений** (tick-scale) — тонкие вертикальные штрихи как на
  измерительной линейке; повторяется в шапке, под баллом, на разделителях.

## 2. Палитра (точные значения — НЕ отклоняться)
```
--void:        #08080a;   /* базовый фон, обсидиан */
--surface:     #101014;   /* панели */
--surface-2:   #16161c;   /* вложенные блоки */
--line:        #24242c;   /* волосяные границы, холодные */
--line-soft:   #1c1c22;
--molten-1:    #ff5312;   /* градиент лавы: старт */
--molten-2:    #ff9a1f;   /* градиент лавы: конец */
--molten-core: #ff6a24;   /* сплошной акцент */
--steel:       #6f9bff;   /* холодная сталь: данные, вторичное */
--steel-dim:   #3c4a6b;
--ash:         #8b8b97;   /* приглушённый текст (холодный серый) */
--parchment:   #ece6dc;   /* тёплый почти-белый для заголовков (НЕ чистый #fff) */
--crit:        #e5484d;   /* светофор: низкий PSQ */
--warn:        #f5a623;   /* светофор: средний */
--good:        #46c48a;   /* светофор: высокий */
```
Правило: чистый белый и чистый серый НЕ использовать. Заголовки — `--parchment`,
тело — `--parchment` при 82% прозрачности, приглушённое — `--ash`.

## 3. Типографика
- **Дисплей (H1, крупные цифры):** тяжёлый гротеск. Стек: `"Segoe UI", system-ui, sans-serif`,
  `font-weight: 800`, `letter-spacing: -0.03em`. H1 — заглавные не нужны; крупно, плотно.
- **Эйробровки/лейблы (над секциями):** `text-transform: uppercase`, `letter-spacing: 0.22em`,
  `font-size: 0.72rem`, цвет `--ash`. Каждая секция начинается с такой метки — это ритм.
- **Тело:** `system-ui`, `line-height: 1.6`.
- **Данные и баллы:** МОНОШИРИННЫЙ — `ui-monospace, "SF Mono", "Cascadia Code", "Roboto Mono",
  monospace`, `font-variant-numeric: tabular-nums`. Все числа/PSQ/счётчики — только моно.

## 4. Сетка, форма, поверхности
- Контейнер `max-width: 1080px`, боковые поля-«поля прибора».
- Углы: панели `border-radius: 14px`; чипы/кнопки `10px`; НЕ круглить всё подряд.
- Границы: 1px `--line`; на активном — молтен. Тени мягкие, низкие: `0 10px 40px -20px #000`.
- Разделители секций — не сплошная линия, а **шкала делений**: ряд тонких `2px×8px` штрихов
  `--line` через `4px` (реализовать через repeating-linear-gradient).

## 5. Кастомные SVG-иконки (ВМЕСТО ЭМОДЗИ)
Все stroke-иконки, `stroke-width:1.6`, `stroke:currentColor`, `fill:none`, `24×24`, скруглённые концы.
Набор (минимум): gauge (аналитика), list-check (история/CSP), flag (накрутка), bar-chart,
share (узел из 3 точек), copy, download, arrow-right/left, triangle-alert (предупреждение),
sparkline. Активное состояние — `currentColor` = молтен; покой — `--ash`. Хранить как inline `<svg>`
или JS-функции-фабрики. НИ ОДНОГО символа эмоджи (⚠️ ✨ 🎉 □ и т.п.) в разметке.

## 6. Движение (детально, «в движении»)
Всё уважает `@media (prefers-reduced-motion: reduce)` — тогда переходы мгновенные.
1. **Загрузка страницы:** секции появляются лестницей — `opacity 0→1` + `translateY(14px→0)`,
   стагер 70мс, ease `cubic-bezier(.2,.7,.2,1)`.
2. **Балл PSQ — герой:** не просто число. Полукруговой **gauge-дуга** (SVG/Canvas): дуга
   заливается от 0 до значения за 900мс (ease-out), одновременно число считается вверх (count-up)
   моноширинным. Цвет дуги — по светофору.
3. **Шкала-подпись:** в шапке под лого — анимированная строка делений с медленным молтен-мерцанием
   (сдвиг градиента 8s, только при motion разрешён).
4. **Вкладки Оценка/Студия:** скользящий подчёркивающий индикатор (transform, 250мс), а не резкая смена.
5. **Ховеры:** панели чуть поднимаются (`translateY(-2px)`) + мягкое молтен-свечение по краю;
   кнопки — световой блик, проходящий слева направо (`::after` sheen, 600мс).
6. **CSP-8 плитки:** при активной — пульс-заполнение (scale 0.96→1 + glow), стагер по индексу.
7. **Появление карточек результата/истории:** fade+rise, стагер.

## 7. Компоненты (переработать ВСЕ по этой системе)
- **Шапка:** слева — вордмарк «SuperPromt» (дисплей, молтен-градиент по тексту) + тонкий
  молтен-слэб (наклонная планка) как логотип-знак; справа — индикатор режима (fast/deep) с
  SVG-точкой (пульс). Под шапкой — мотив шкалы делений во всю ширину.
- **Ввод:** панель `--surface`, textarea `--void`, чипы-примеры как «клавиши прибора».
- **Кнопки:** primary — молтен-градиент + sheen; secondary — контур `--line`, текст `--parchment`.
- **Результат PSQ:** слева gauge-дуга + балл (моно) + вердикт словами; справа сетка CSP-8 плиток;
  ниже — флаги как «предупреждающие бирки» (SVG triangle-alert + моно-счётчик).
- **Аналитика/История/Студия:** те же поверхности, эйробровки-метки, SVG-иконки заголовков,
  моноширинные числа, шкала-разделители.

## 8. Анти-паттерны (ЧЕГО НЕ ДЕЛАТЬ — это «вайб-кодинг»)
- Эмоджи как иконки; чистый `#fff`/`#000`; повсеместный `border-radius: 16px`; центрирование всего;
  один фиолетово-синий градиент; Inter/Space Grotesk «по умолчанию»; тень-ободок на скруглённых
  карточках без иерархии; статичный интерфейс без motion. Ничего из этого.
