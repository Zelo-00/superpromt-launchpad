О проекте SuperPromt

SuperPromt автоматизирует постановку задач по метрике PSQ.

Режимы:

  • Оценка — быстрый детерминированный анализ.
  • Студия — ручная доводка по CSP-8.
  • Сравнение — A/B тестирование промтов.
  • Аналитика — статистика и тренды.

Фишка — анти-накрутка гейт ДО траты токенов LLM.
