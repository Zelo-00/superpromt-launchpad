# Knowledge Work — универсальный пак интеллектуальной работы

Источник: https://github.com/anthropics/knowledge-work-plugins ·
локальная копия: `~/metodika/SuperPromt/vendor/knowledge-work-plugins/` (использовать её: офлайн, проверена).

## Когда применять
Задача из офисного/предметного домена: финансовый анализ и модели, юридические документы и
ревью договоров, продажи (письма, CRM-гигиена, квалификация), продукт (PRD, роадмап, интервью),
маркетинг (позиционирование, контент-план), аналитика данных (без ML-исследования), HR, поддержка.
НЕ применять: чистая разработка (→ код-скиллы), научные доказательства (→ math-orchestr).

## Цепочка
1. Определи поддомен → возьми соответствующий плагин из пака (finance / legal / sales /
   product-management / marketing / data / hr / support / research).
2. Собери контекст: исходники, ограничения, целевая аудитория результата, формат выхода.
3. Выполни по практике поддомена из плагина (шаблоны, чеклисты, структура документа).
4. Факт-контроль: числа пересчитать, ссылки/источники проверить (гейт выхода), допущения пометить.
5. Выход в запрошенном формате (документ/таблица/письмо/презентация — для слайдов → frontend-slides).
