# Frontend Slides — создание презентаций (HTML-first)

Источник: https://github.com/zarazhangrui/frontend-slides.

## Когда применять
Презентация/доклад/питч/защита проекта; технический deck с кодом и диаграммами;
слайды из markdown-контента. НЕ применять: длинные документы (→ documents), дашборды.

## Цепочка создания
1. **Нарратив**: одна мысль = один слайд; каркас «проблема → решение → доказательство → призыв»;
   10/20/30-дисциплина (мало слайдов, крупный текст).
2. **Каркас**: HTML-слайды (section на слайд) или reveal.js/Slidev-подход из markdown;
   16:9, сетка, безопасные поля.
3. **Типографика**: заголовок ≤ 2 строк, кегль заголовка ≥ 2× текста, максимум 5 строк текста
   на слайд; контраст ≥ 4.5:1.
4. **Код-слайды**: подсветка синтаксиса, ≤ 15 строк, выделение ключевых строк.
5. **Визуал**: диаграммы (mermaid), фоны (paper-shaders умеренно), единая палитра бренда.
6. **Переходы**: скромные (fade/slide), навигация клавиатурой, прогресс-бар.
7. **Экспорт**: печать в PDF (@media print), автономный HTML-файл одним артефактом.
